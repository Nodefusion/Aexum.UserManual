# Activity Logs

Some resources (like Assets, Vendors, etc.) inside Environment supports Activity Logs feature that is system-wide logs that track various actions and events across the entire Aexum platform, including user activities, system changes, and security-related events. Activity Logs provide a comprehensive overview of what is happening within the system for security and compliance purposes.

Activity Logs are created on each record create, update or delete and is recording:

* Timestamp - date and time when change was made
* Entity - such as Asset, Vendor, Location etc.
* Entity Id - unique ID of each record
* Changed by - user who made the change
* Change Type - Added, Modified, Soft Deleted
* Old Values - previous values of fields that were changed
* New Values - previous values of fields that were changed

You can also export activity log data.

Audit Logs are available in Organization Admin and Environment Admin sections depending on the scope of the resource.
