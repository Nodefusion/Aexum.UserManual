# Organization Roles

Organization Roles are managed similarly to Environment Permission Roles, but they apply at the organization level.
Organization Roles can not be customized, only built-in roles are available.

## Organization Permission Roles

The following built-in Organization Permission Roles are available:

> TODO

| Role Name | Description |
| -- | -- |
| **Organization Owner** | Full access to all organization-level settings and configurations, including user management, billing, and organization-wide policies. |
| **Organization Administrator** | Manage organization settings, users, and permissions, but cannot access billing information. |
